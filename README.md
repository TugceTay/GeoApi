# GeoApi
An interactive geospatial application using ASP.NET Core 6, PostGIS, and Angular 15. Features include creating, reading, updating, and deleting geospatial data and visualizing it on a Leaflet map.

## Overview
MyGeoApp is a dynamic geospatial application that leverages ASP.NET Core 6.0 with an N-Tier architecture for the backend, PostGIS for the geospatial database, and Angular 15 for the frontend. It provides a user-friendly platform for creating, reading, updating, and deleting (CRUD) geospatial data stored in PostGIS and visualizing it on a Leaflet map.

### Technologies Used
* ASP.NET Core 6.0
* PostGIS
* Angular 15
* Leaflet 1.9.3
* Bootstrap 4.5.3
* JQuery 3.6.4
* TypeScript 4.9.4
